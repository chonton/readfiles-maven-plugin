readfiles:readfiles

Full name:

org.honton.chas:readfiles-maven-plugin:0.1.0:readfiles

Description:

Read files into maven properties. The contents of each file is read in full. The content is then assigned to a maven property of the same name.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: initialize.

Optional Parameters

Name Type Since Description
<encoding> String - Charset encoding of the source files. Defaults to UTF-8
Default value is: ${project.build.sourceEncoding}.
<files> File[] - List of files to read
<prefix> String - Prefix for the target properties
User property is: readfiles.prefix.
<regexReplacements> RegexReplacementRule[] 0.1.0 Transformations to apply to file contents before storing it into the property.
<skip> boolean - Skip executing this plugin
Default value is: false.
User property is: readfiles.skip.
<trim> boolean 0.0.2 Whether to trim leading and trailing whitespace characters.
Default value is: false.
User property is: readfiles.trim.
